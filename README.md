# Java Data Structures and Algorithms (DSA)

A comprehensive collection of Java-based implementations of fundamental data structures and algorithms, featuring solutions to popular interview problems from platforms like LeetCode and GeeksforGeeks.

## 🚀 Getting Started

### Prerequisites
*   **Java Development Kit (JDK):** Version 11 or higher is recommended.
*   **IDE:** IntelliJ IDEA, Eclipse, or VS Code with Java extensions.

### Installation
1.  Clone the repository:
    ```bash
    git clone https://github.com
    ```
2.  Navigate to the project directory:
    ```bash
    cd Java-DSA
    ```

## 📂 Repository Structure

The project is organized by topic for easy navigation:
*   `/Arrays` - Sliding window, two pointers, and sorting.
*   `/LinkedLists` - Singly, doubly, and circular linked lists.
*   `/Trees` - Binary trees, BSTs, and heaps.
*   `/Graphs` - BFS, DFS, and shortest path algorithms.
*   `/Recursion_DP` - Classic recursive problems and dynamic programming.

## 🛠️ Usage

To run a specific solution, compile and execute it using the terminal:
```bash
javac Arrays/TwoSum.java
java Arrays.TwoSum
